材料
レンズ豆 100g （柔らかくなるまで煮て、煮汁はとっておく）
菜の花 1/4束 （細かく刻む）
ほうれん草 1/4束 （細かく刻む）
香菜 1束 （細かく刻む）
干ししいたけ 3枚 （水で戻して薄切りに）
玉ねぎ 1個
にんにく 1片 それぞれをみじん切りにするか、合わせてフードプロセッサーにかける
しょうが 1片
オリーブオイル 大さじ3〜4
ホールトマト缶 400ml
カレー粉 大さじ１ 1/2
ヨーグルト 大さじ3
唐辛子 2本
塩 小さじ1
乾物スープ 400ml
 
作り方
1. 厚手の鍋にオリーブオイル大さじ２を熱し玉ねぎ、にんにく、しょうがを加えて20分炒める。
2. 1にトマト缶を加えカレー粉、ヨーグルト、唐辛子、乾物スープ、塩を加えて15分程煮込む。
3. フライパンにオリーブオイル大さじ１を熱し菜の花、ほうれん草と干ししいたけを炒め、 2にレンズ豆と香菜を加え、塩で味を整えて30分程煮込む。
